##주의사항##
1. 일대다 양방향보단 다대일 양방향을 사용하자.
2. 데이터베이스 테이블, 컬럼명 : 소문자 + _ 조합 스타일로 사용한다. (각자 스타일)
3. 외래키가 있는 곳을 연관관계의 주인으로 정하자. N : 1이면 N이 외래키가 있다.
